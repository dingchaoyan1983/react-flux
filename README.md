## build:
### CLI Mode:
* `npm install -g browserify`
* `npm install`
* `browserify js/app.js > dist/app.bundle.js`
* `browserify browserify js/fixtures/main.js > dist/fixture.bundle.js`

### Or Gulp Mode:

* `npm install -g gulp`
* `npm install`
* `gulp`

### Or Webpack Mode:

* `npm install -g webpack`
* `npm install`
* `webpack`
